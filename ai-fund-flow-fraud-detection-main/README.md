# 🚀 GraphSentinel Lite

AI-powered real-time fraud detection system using:

## 🔥 Tech Stack
- Kafka (Real-time streaming)
- Node.js + Express
- MongoDB
- React + Next.js
- TensorFlow.js (AI Model)
- Socket.IO (Live alerts)

## 🧠 Features
- Real-time fraud detection
- Graph-based fund flow visualization
- AI + rule-based hybrid scoring
- Analyst feedback learning system
- Explainable AI (reasons for fraud)
- Investigator mode (trace transactions)
- Demo mode (simulate fraud scenarios)

## ⚡ How it works
1. Transactions → Kafka
2. Fraud engine analyzes
3. AI model predicts risk
4. Alerts pushed via Socket.IO
5. Analyst feedback improves model
